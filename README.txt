This is a student project focusing on front-end web design.

The project utilizes HTML and CSS to display an attractive and functional front-end design for a Dungeons and Dragons 5E informational website,
a variety of HTML elements are presented, as well as several different media types and CSS styling.

There is no backend or database, and no data processing occurs. This was meant to be an exercise in front-end design only.

website link: https://pepper37.github.io/world-of-dnd/
